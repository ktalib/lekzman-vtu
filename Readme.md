### Core Functionalities

## Usage ğŸ“±

1. **User Registration/Login**
   - Create an account or login
   - Complete profile setup

2. **Electricity Payments**
   - Select electricity provider
   - Enter meter number
   - Specify amount
   - Complete payment

3. **VTU Top-up**
   - Choose network provider
   - Enter phone number
   - Select amount
   - Process payment
